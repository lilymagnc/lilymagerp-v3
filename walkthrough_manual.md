# 사용자 매뉴얼 고도화 및 콘텐츠 복구 완료 (Walkthrough)

## 프로젝트 성과
LilyMag ERP 시스템의 **사용자 매뉴얼(`src/app/dashboard/manual/page.tsx`)**을 성공적으로 복구하고, 현대적인 UI 가이드와 풍부한 콘텐츠로 고도화하였습니다.

## 주요 변경 사항

### 1. 콘텐츠 체계 재정립 ("제 N장" 스타일)
- 기존의 단순 나열 방식에서 벗어나 **"제 1장. 대시보드"**부터 **"제 22장. 인사/사용자/설정"**까지 체계적인 챕터 기반 구조를 도입했습니다.
- 모든 제목 섹션에 아이콘과 구분선을 배치하여 가독성을 극대화했습니다.

### 2. 시각적 가이드(UI Mockups) 강화
- **`BrowserFrame`**: 실제 시스템 화면을 브라우저 창 안에 있는 것처럼 묘사하여 사용자 이해도를 높였습니다.
- **`FeatureCard`**: 주요 포인트와 강조 기능을 아이콘과 함께 박스 형태로 배치했습니다.
- **`StepGuide`**: 따라하기 쉬운 단계별 안내 UI를 적용했습니다.
- **`Highlight`**: 텍스트 중 중요한 키워드(버튼 명칭 등)를 강조 처리했습니다.

### 3. 역할 기반(RBAC) 콘텐츠 노출 최적화
- `isHQ` 변수를 활용하여 **본사 관리자**와 **지점 사용자**에게 각각 필요한 메뉴와 설명만 표시되도록 정교하게 분기 처리했습니다.
- 사이드바 내비게이션도 사용자 권한에 따라 동적으로 구성됩니다.

### 4. 인터랙티브 기능 개선
- **사이드바 내비게이션**: 클릭 시 해당 챕터로 부드럽게 스크롤되는 기능을 구현했습니다.
- **Scroll Spy**: 현재 읽고 있는 섹션을 감지하여 사이드바의 활성 상태를 실시간으로 업데이트합니다.
- **배지(Badge) 시스템**: 현재 접속 권한을 상단 헤더에 명확히 표시합니다.

## 상세 업데이트 내역 (총 22개 장)

- **제 1-3장**: 대시보드 요약, 일정/주문 연동, 매장 운영 체크리스트 및 템플릿 안내
- **제 4장**: 태그 기반 샘플앨범 및 제작 완료 사진 연동 가이드
- **제 5장**: PC와 Mobile 각 플랫폼에 최적화된 주문 접수 프로세스
- **제 6-9장**: 실시간 주문 모니터링, 이관(Transfer), 외부 발주, 픽업/배송 관리 및 배송 사진 전송
- **제 10-13장**: CRM 고객 등급 관리, 거래처/업체 관리, 견적-주문 연동, 비대면 인사 서류 신청
- **제 14-22장 (본사 전용)**: 전 지점 상품 일괄 제어, 재고 변동 로그, 자재 요청 승인, 통합 비용/예산 관리, 리포트 성과 분석 등

## 향후 관리 가이드
- 새로운 메뉴가 추가될 경우 `menuSections` 배열에 ID와 라벨을 추가하고, 본문에 해당 ID를 가진 `<section>`을 생성하면 자동으로 사이드바와 스크롤 스파이에 반영됩니다.
- 본문의 모든 설명은 한국어로 작성되었으며, 공신력 있는 용어를 사용하였습니다.
